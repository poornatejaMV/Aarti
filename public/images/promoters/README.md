# Promoters Images

This directory should contain images for the promoters/team members.

## Required Images

Please add the following image files:

1. `pramod-khairnar.jpg` - Image of Shri Pramod Khairnar (Vice President)
2. `t-r-jadhav.jpg` - Image of Wing Commander T. R. Jadhav (Secretary, Retired Military)
3. `sunil-kirdak.jpg` - Image of Shri Sunil Kirdak (Secretary)
4. `anuradha-chavan.jpg` - Image of Smt. Anuradha Chavan (Secretary)
5. `satish-tupe.jpg` - Image of Shri Satish Tupe (Secretary)

## Image Specifications

- Recommended size: 400x400px or larger
- Format: JPG or PNG
- Aspect ratio: Square (1:1) preferred
- The images will be automatically optimized by Next.js Image component

## Placeholder

If images are not available, the components will attempt to use `/images/placeholder-person.jpg` as a fallback. You can create a generic placeholder image if needed.
